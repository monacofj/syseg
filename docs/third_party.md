Third-party artifacts shipped with this project
===========================================

The following third-party artifacts are shipped with this project for
convenience.

CSM Wrapper
-----------

CSMWrap is an EFI application designed to enable legacy BIOS booting on modern
UEFI-only (class 3) systems by wrapping a Compatibility Support Module (CSM)
build of the SeaBIOS project as an out-of-firmware EFI application.

* `tools/csmwrapia32.efi`  
* `tools/csmwrapx64.efi`

https://github.com/CSMWrap/CSMWrap
