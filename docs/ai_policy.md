# AI usage disclosure and intellectual integrity

Auxiliary AI assistance tools have been used in this project under the authors' 
oversight for technical tasks such as repository maintenance chores, defect 
detection, code compliance auditing, test suite automation, and refinement of 
documentation. The human authors remain fully responsible for the intellectual 
work and in effective control of the design and implementation of the software 
functionality, architectural decisions, and overall codebase development.

Contributors must disclose any AI usage beyond trivial assistance. Submissions 
of code generated exclusively by AI without human oversight are not allowed, 
as all contributions must reflect the authors' own intellectual engagement and 
accountability.
